动手跑一个完整的 L2 交换机实验。

> 本文基于 [P4 中文教程](https://github.com/zhh2001/p4-language-guide-zh) examples/02-l2-switch 整理。

---

## 一、实验目标

实现一个简单的 L2 交换机，根据目的 MAC 地址转发。

---

## 二、核心 table

```p4
table l2_table {
    key = { hdr.ethernet.dstAddr: exact; }
    actions = { set_egress; broadcast; }
    default_action = broadcast();
}
```

---

## 三、运行实验

```bash
cd examples/02-l2-switch
sudo bash run.sh
```

---

## 四、手动配置表项

```bash
simple_switch_CLI --thrift-port 9090
s1> table_add l2_table set_egress 00:00:00:00:00:01 => 1
```
