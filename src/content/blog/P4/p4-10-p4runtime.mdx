## 一、P4Runtime 工作流程

1. 读取 P4Info
2. 选择主控制器
3. 推送 pipeline
4. 读写表项
5. 收发 Packet-in/Packet-out

---

## 二、Python 控制平面

```python
from p4runtime_lib.bmv2 import Bmv2Connection

conn = Bmv2Connection(
    device_id=0,
    grpc_addr='localhost:50051',
    election_id=(0, 1)
)
conn.setup()
```
