概念讲完了，现在动手搭建环境。P4 工具链包含编译器（p4c）、软件交换机（BMv2）、拓扑仿真（Mininet）等组件。

---

## 一、方案选择

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| Docker | 环境隔离，一键部署 | 镜像约 3GB | 推荐 |
| apt | 一条命令装完 | 仅支持 Ubuntu | 旧版本 |
| 源码编译 | 最新特性 | 编译 30-60 分钟 | 开发者 |

---

## 二、Docker 方式（推荐）

### 步骤 1：安装 Docker

```bash
# Ubuntu
sudo apt update
sudo apt install docker.io
sudo usermod -aG docker $USER
# 重新登录
```

### 步骤 2：拉取镜像

```bash
docker pull p4lang/p4app:latest
```

### 步骤 3：启动容器

```bash
docker run -it --privileged --name p4test p4lang/p4app:latest bash
```

### 步骤 4：验证

```bash
p4c --version
simple_switch --version
```

---

## 三、验证环境

```bash
# 检查 p4c
p4c --version

# 检查 BMv2
simple_switch --version

# 检查 Mininet
sudo mn --version
```

---

## 四、常见问题

| 问题 | 解决方案 |
|------|----------|
| Docker 权限不足 | `sudo usermod -aG docker $USER` 后重新登录 |
| 镜像拉取慢 | 使用国内镜像源 |
| 编译报错 | 检查 P4 文件语法 |
