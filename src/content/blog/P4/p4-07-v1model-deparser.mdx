V1Model 是 BMv2 的参考架构，定义了报文处理的完整流水线。

> 本文基于 [P4 中文教程](https://github.com/zhh2001/p4-language-guide-zh) 第 09、11 章整理。

---

## 一、V1Model 架构

```
入口 → Parser → Ingress → Egress → Deparser → 出口
```

| 块 | 作用 |
|----|------|
| `Parser` | 解析报文 |
| `VerifyChecksum` | 校验和验证 |
| `MyIngress` | 入口处理 |
| `MyEgress` | 出口处理 |
| `ComputeChecksum` | 计算校验和 |
| `Deparser` | 重新打包 |

---

## 二、Deparser

Deparser 的作用是把处理好的 header 按顺序输出：

```p4
control MyDeparser(packet_out pkt, in Headers hdr) {
    apply {
        pkt.emit(hdr.ethernet);
        pkt.emit(hdr.ipv4);
    }
}
```

---

## 三、standard_metadata 关键字段

| 字段 | 作用 |
|------|------|
| `ingress_port` | 入端口号 |
| `egress_spec` | 指定出端口 |
| `packet_length` | 报文长度 |
