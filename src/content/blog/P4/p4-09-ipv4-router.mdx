实现一个 IPv4 路由器，支持跨子网转发。

> 本文基于 [P4 中文教程](https://github.com/zhh2001/p4-language-guide-zh) examples/03-ipv4-router 整理。

---

## 一、核心 table

```p4
table ipv4_lpm {
    key = { hdr.ipv4.dstAddr: lpm; }
    actions = { set_nhop; drop; }
}
```

---

## 二、路由表配置

```bash
s1> table_add ipv4_lpm set_nhop 10.0.1.0/24 => 1 10.0.1.1
s1> table_add ipv4_lpm set_nhop 10.0.2.0/24 => 2 10.0.2.1
```
